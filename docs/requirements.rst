============
Requirements
============

:abbr:`AOIC (ANOPP2 OpenMDAO Interface Code)` requires the installation of OpenMDAO in addition to Python 2.7 along with the numerical and scientific packages NumPy and SciPy. AOIC is a Python plugin in OpenMDAO that the user will have to create to use ANOPP2 for noise optimization. AOIC also requires library, ``libANOPP2.so``, and the ANOPP2 API, ``ANOPP2_api.py``.
